# 🖼️ Image Vertical Splitter（画像縦4分割ツール）

アップロードした画像を縦方向に4等分するシンプルなWebアプリケーションです。

## 機能

- **ドラッグ＆ドロップ対応** - クリックまたはD&Dで画像をアップロード
- **プレビュー表示** - 元画像を確認してから分割
- **縦4分割** - 画像を上から順に4つのパートに分割
- **個別ダウンロード** - 各パートを個別にPNG形式でダウンロード
- **一括ダウンロード** - すべてのパートをまとめてダウンロード

## 対応画像形式

- JPG / JPEG
- PNG
- GIF
- WebP

## 使い方

1. `index.html` をブラウザで開く
2. 画像をドラッグ＆ドロップ、またはクリックしてファイルを選択
3. プレビューを確認し「4分割する」ボタンをクリック
4. 分割された画像を個別または一括でダウンロード

## ファイル構成

```
image-splitter/
├── index.html    # メインHTML
├── style.css     # スタイルシート
├── script.js     # 画像処理ロジック
└── README.md     # このファイル
```

## 技術仕様

- **HTML5 Canvas API** を使用して画像を分割
- フレームワーク不要、純粋なHTML/CSS/JavaScript
- モダンなダークテーマUI
- レスポンシブデザイン対応
