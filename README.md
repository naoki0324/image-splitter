# 🖼️ Image Vertical Splitter（画像縦4分割ツール）

アップロードした画像を縦方向に4等分するシンプルなWebアプリケーションです。

## 機能

- **ドラッグ＆ドロップ対応** - 画像をドロップゾーンにドラッグするだけでアップロード
- **クリックで選択** - ドロップゾーンのクリックまたは「ファイルを選択」ボタンからアップロード
- **プレビュー表示** - 元画像と分割結果を並べて確認
- **縦4分割** - 画像を上から順に4つのパートに自動分割
- **個別ダウンロード** - 各パートを個別にPNG形式でダウンロード
- **一括ダウンロード** - すべてのパートをまとめてダウンロード
- **エラーハンドリング** - 読み込み失敗時にアラートで通知

## 対応画像形式

- JPG / JPEG
- PNG
- GIF
- WebP
- その他ブラウザがサポートする画像形式

## 使い方

1. ローカルサーバーを起動（下記参照）
2. ブラウザで `http://localhost:8080` を開く
3. 画像をドラッグ＆ドロップ、またはクリックしてファイルを選択
4. 自動的に4分割された結果が表示される
5. 分割された画像を個別または一括でダウンロード

### ローカルサーバーの起動方法

```bash
cd image-splitter
python3 -m http.server 8080
```

> **Note**: `file://` プロトコルではダウンロード機能が制限されるため、ローカルサーバー経由での使用を推奨します。

## 技術仕様

- **HTML5 Canvas API** を使用して画像を分割
- **Blob API** を使用して確実なダウンロードを実現
- フレームワーク不要、純粋なHTML/CSS/JavaScript
- シンプルで落ち着いたライトテーマUI
- レスポンシブデザイン対応（モバイル・タブレット対応）
